# Trained Models

This folder contains the trained Keras model files used in the project.

## Models

- `simple_cnn_model.keras`: Custom CNN baseline model.
- `vgg16_model.keras`: Transfer learning model based on VGG16.
- `resnet50_model.keras`: Transfer learning model based on ResNet50.

## Notes

The models were trained to classify breast cancer histopathology images into:

- Benign
- Malignant

If the model files are not included because of GitHub file size limits, they can be regenerated by running the notebook.
